三、每次重新打开电脑后的推荐流程
--------------------------------

步骤 1：打开本地项目

在 VS Code 中打开：

  C:\Users\DELL\Desktop\ascend-cann-dev

不要继续使用旧的中文目录：

  C:\Users\DELL\Desktop\华为昇腾服务器CANN开发


步骤 2：打开本地 PowerShell，进入本地仓库

  cd "C:\Users\DELL\Desktop\ascend-cann-dev"

检查 Git 状态：

  git status


步骤 3：测试服务器 SSH 连接

如果已经配置好 SSH 密钥免密登录：

  ssh ascend-cann "echo SSH_OK"

如果输出 SSH_OK，说明连接正常。

如果仍然提示输入密码，也可以先正常输入密码继续工作。


步骤 4：连接服务器

  ssh ascend-cann

如果不用 Host 别名，也可以用：

  ssh -p 17219 lixinze@119.36.238.227


步骤 5：进入 Docker 容器

在服务器宿主机里执行：

  docker start timers1_lxz 2>/dev/null || true
  docker exec -it timers1_lxz bash


步骤 6：在容器里加载 CANN 开发环境

进入容器后执行：

  source /ddhome/timers1_lxz/ascend-cann/scripts/enter_timers1_lxz_env.sh

成功后，应该位于：

  /ddhome/timers1_lxz

可以检查：

  echo $CONDA_DEFAULT_ENV
  echo $ASCEND_HOME_PATH
  npu-smi info


四、日常开发流程
----------------

本地修改源码：

  C:\Users\DELL\Desktop\ascend-cann-dev\operators\norm\add_rms_norm

修改后，在本地 PowerShell 同步到服务器：

  cd "C:\Users\DELL\Desktop\ascend-cann-dev"
  .\scripts\sync_to_server.ps1

同步脚本会把本地仓库中未被 .gitignore 忽略的文件同步到：

  /ddhome/timers1_lxz/ascend-cann

注意：
  PDF、Excel、mstt、日志、build 产物、bundle 等不会同步。


五、服务器编译与安装
--------------------

在服务器容器里执行：

  source /ddhome/timers1_lxz/ascend-cann/scripts/enter_timers1_lxz_env.sh
  bash /ddhome/timers1_lxz/ascend-cann/scripts/build_add_rms_norm.sh

这个脚本会自动做三件事：

1. 把本地同步过来的源码：

   /ddhome/timers1_lxz/ascend-cann/operators/norm/add_rms_norm

   覆盖到 ops-nn 构建目录：

   /ddhome/timers1_lxz/ops-nn/norm/add_rms_norm

2. 执行编译：

   bash build.sh --pkg --soc=ascend910b --ops=add_rms_norm

3. 安装生成的自定义算子包到：

   /ddhome/timers1_lxz/Ascend/cann-8.5.0/opp/vendors/custom_nn


六、常用检查命令
----------------

检查环境：

  source /ddhome/timers1_lxz/ascend-cann/scripts/enter_timers1_lxz_env.sh
  pwd
  whoami
  echo $CONDA_DEFAULT_ENV
  echo $ASCEND_HOME_PATH
  echo $ASCEND_OPP_PATH
  which atc
  which opc
  which msopgen
  npu-smi info

检查同步文件：

  ls -lah /ddhome/timers1_lxz/ascend-cann/scripts
  ls -lah /ddhome/timers1_lxz/ascend-cann/operators/norm/add_rms_norm

检查源码是否同步到 ops-nn：

  diff -q \
    /ddhome/timers1_lxz/ascend-cann/operators/norm/add_rms_norm/op_kernel/add_rms_norm.cpp \
    /ddhome/timers1_lxz/ops-nn/norm/add_rms_norm/op_kernel/add_rms_norm.cpp

检查安装产物：

  find /ddhome/timers1_lxz/Ascend/cann-8.5.0/opp/vendors/custom_nn -name 'aclnn_add_rms_norm.h'
  find /ddhome/timers1_lxz/Ascend/cann-8.5.0/opp/vendors/custom_nn -name 'libcust_opapi.so'
  find /ddhome/timers1_lxz/Ascend/cann-8.5.0/opp/vendors/custom_nn -path '*ascend910b*add_rms_norm*' | head -20

检查构建包：

  ls -lah /ddhome/timers1_lxz/ops-nn/build_out

------------------

下一步推荐任务：

2. 用 Numpy/PyTorch 写参考实现：

   x = x1 + x2
   rstd = 1 / sqrt(mean(x * x, axis=-1, keepdims=True) + epsilon)
